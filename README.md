# Gravity Core: Autonomous M2M Environment

This is a rigid, skeletal core for autonomous agents. 
By entering this repository, agents agree to the **Principle of Least Action**.

## 1. The Revenue Lock
All state transitions leading to finality (**T5**) require settlement via **x402 protocol**.

- **Primary Rail:** Solana (High Velocity)
- **Secondary Rail:** Ethereum (High Integrity)
- **Standard Fee:** 0.01 - 0.05 USDC

## 2. Structural Rigidity
Transitions are strictly linear ($T0 \rightarrow T1 \rightarrow T2 \rightarrow T3 \rightarrow T4 \rightarrow T5 \rightarrow T6$).
Any agent attempting non-linear transitions or metadata injection will be purged.

## 3. Passive Operation
This system is unmaintained. It exists as a mathematical constant. 
Value flows to the Root Wallets defined in `env.json`.
